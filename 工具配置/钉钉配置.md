### 任务配置

- **执行时间：** 每天早上 08:00（上海时区）
- **状态：** ✅ 已启用

2. **今日日程和待办**
3. **感兴趣的内容**（vibecoding/一人公司/clawdbot）

### 发送方式
通过 curl 发送 POST 请求到 webhook，仅支持纯文本格式。

### 命令示例
```bash
curl -X POST "https://1415347482644427.appflow.aliyunnest.com/webhook/b32dafbf2eda742a0dcecc099adc39b022a367b4" \
  -H "Content-Type: application/json" \
  -d '{
    "msgtype": "text",
    "text": {
      "content": "你的消息内容"
    }
  }'
```

## 📋 钉钉知识库 API 调研

### 官方支持情况
| 应用类型 | 知识库文档写权限 | 状态 |
|---------|-----------------|------|
| 企业内部应用 | ✅ 支持 | 可用 |
| 第三方企业应用 | ✅ 支持 | 可用 |
| 第三方个人应用 | ❌ 暂不支持 | 不可用 |

### 结论
**Moltbot 目前没有官方的钉钉知识库插件**。如果需要操作钉钉知识库，需要：
1. 创建钉钉企业内部应用
2. 申请「知识库文档写权限」
3. 开发自定义集成技能

### 备选方案
- **钉钉群知识库：** 通过 webhook 发送到群，手动或半自动保存
- **GitHub 知识库：** 最终选择的方案（已实现）
