## 📌 项目记录

### Pixso MCP 方案调研
- **日期：** 2026-02-03
- **目的：** 探索 Figma 转 React 代码的新方案
- **发现：** Pixso MCP 是 Model Context Protocol 协议，通过 AI 编程助手（如 Cursor）实时读取设计并生成代码
- **对比：** 相比传统 Figma-to-Code 工具，MCP 方案能生成更符合项目规范的代码
- **流程：** Pixso 客户端 → MCP 服务器 → AI 编程助手 → 代码

### Figma 转 React 工具对比
调研了多个免费/付费方案：
1. **Locofy.ai** - 推荐，有免费额度
2. **Builder.io** - 学习能力强，生产级代码
3. **PlatUI** - 完全免费，Figma 插件
